## 项目结构
```
└── src
    ├── assets         资源文件
    ├── components     公用组件
    ├── request        请求
    ├── router         路由
    ├── store          状态管理
    └── views
        ├── Login      登录
        ├── News       新闻
        ├── Panel      个人面板
        ├── Register   注册
        ├── Supplier   供应商
├── vue.config         vue自定义配置
├── .eslintrc          eslint规则配置
```


## 路由结构
```shell
└── /panel             个人面板-需登录
    ├── my-order       资源文件
├── /login             登录
├── /register          注册
├── /news              新闻列表
├── /supplier          供应商招标
```
